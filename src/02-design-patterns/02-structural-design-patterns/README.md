# Structural Design Pattern

- Family of design patterns which help in dealing with object composition or structure of classes.
- They ensure that if one part of a system changes, the entire system doesn't need to do the same.
- These make the system more flexible and easier to maintain.

## Types of Structural Design patterns

- Adapter pattern
- Bridge pattern
- Composite pattern
- Decorator pattern
- Facade pattern
- Flyweight pattern
- Proxy pattern
